# Geração Tech - Landing Page

![Banner do Projeto](https://img.shields.io/badge/Status-Conclu%C3%ADdo-brightgreen?style=for-the-badge)
![License](https://img.shields.io/badge/License-MIT-blue?style=for-the-badge)

Uma **Landing Page** moderna e responsiva desenvolvida como atividade do curso **Geração Tech 2.0**. O projeto foca em uma experiência visual imersiva, utilizando elementos de vídeo em background e uma interface refinada para apresentar serviços e soluções tecnológicas.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as melhores práticas de desenvolvimento Web Front-end:

*   ![HTML5](https://img.shields.io/badge/html5-%23E34F26.svg?style=for-the-badge&logo=html5&logoColor=white) - Estruturação semântica.
*   ![CSS3](https://img.shields.io/badge/css3-%231572B6.svg?style=for-the-badge&logo=css3&logoColor=white) - Estilização avançada e Flexbox/Grid.
*   ![JavaScript](https://img.shields.io/badge/javascript-%23323330.svg?style=for-the-badge&logo=javascript&logoColor=%23F7DF1E) - Interatividade e manipulação de DOM.
*   **Vídeo Background** - Para uma experiência de usuário dinâmica e moderna.

---

## 📁 Estrutura do Projeto

A organização de arquivos segue o padrão de clareza e manutenibilidade:

```text
├── assets/             # Mídias (Imagens, Logos e Backgrounds em vídeo)
├── css/                # Folhas de estilo (style.css)
├── js/                 # Lógica e interatividade (main.js)
├── favicon.png         # Ícones do navegador
└── index.html          # Ponto de entrada principal
```

---

## ✨ Funcionalidades Principais

- [x] **Background Animado:** Uso de vídeo (`background.mp4`) para captar a atenção do usuário instantaneamente.
- [x] **Design Responsivo:** Adaptado para dispositivos móveis, tablets e desktops.
- [x] **Seção de Portfólio/Serviços:** Exibição clara de competências através de cards visuais.
- [x] **Navegação Fluida:** Menu interativo e scroll suave para melhor UX.

## 🚀 Como Executar o Projeto

Como este é um projeto baseado em tecnologias nativas (HTML/CSS/JS), não é necessário instalar dependências pesadas.

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/flaviare1s/landing-page-geracao-tech.git
    ```

2.  **Acesse a pasta do projeto:**
    ```bash
    cd landing-page-geracao-tech
    ```

3.  **Abra o arquivo principal:**
    Basta abrir o arquivo `index.html` em seu navegador de preferência ou utilizar a extensão **Live Server** no VS Code para visualizar em tempo real.
